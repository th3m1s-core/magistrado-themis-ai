## 1. Descrição do Invento
O sistema compreende um pipeline de dados que inicia na captura de microtexturas ópticas via hardware (GuardTag) e culmina na geração de um laudo jurídico via LLM.

## 2. Arquitetura "Themis-Engine"
- **Input:** JSON UEAP v1.0.
- **Processamento:** Inferência local (Edge Computing) via Qwen-3B optimizada para termos forenses.
- **Contextualização:** Injeção de "Legal Prompting" baseado no Hexágono Legislativo (LPI, LGPD, etc.).
- **Output:** Laudo Assinado (`.pdf` ou `.json` assinado).

## 3. Estado da Técnica (Benchmarking)
Sistemas atuais (ex: Certificadoras) dependem de análise humana manual para emitir laudos. O Magistrado Themis elimina o fator de erro humano e o delay temporal da auditoria.
